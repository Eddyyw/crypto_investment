# Crypto investment

## Requirements
* Ruby 3.0.6
* Rails 6.1.7.6
* Postgresql 15

## Instalation instructions

### Clone repository
```bash
git clone git@github.com:Eddyyw/crypto_investment.git
```

### Install gems
```bash
bundle install
```

## Starting the application
```bash
bin/rails s # will start rails server
```

## Starting a console

```bash
bin/rails console
```
